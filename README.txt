1. Κάνε αποσυμπίεση.
2. Άνοιξε terminal μέσα στον φάκελο flipbook.
3. Τρέξε: python -m http.server 8000
4. Άνοιξε: http://localhost:8000

Η έκδοση αυτή χρησιμοποιεί online βιβλιοθήκη flipbook, άρα χρειάζεται internet.
Περιλαμβάνει το δικό σου MP3 για ήχο γυρίσματος σελίδας.
Στην τελευταία σελίδα, το Επόμενο επιστρέφει στην πρώτη.
